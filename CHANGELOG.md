# Changelog

Все значимые изменения в проекте Prime Messenger будут задокументированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [Невыпущенное]

### Добавлено
- ✅ Системные уведомления при новых сообщениях
- ✅ Счётчик непрочитанных сообщений в списке чатов
- ✅ Звуковое сопровождение уведомлений
- ✅ Обновление счётчика через SignalR в реальном времени
- ✅ Toast-уведомления с информацией о сообщении
- ✅ Иконки и badge для системных уведомлений
- ✅ Сброс счётчика при открытии чата

### Изменено
- ✅ Улучшена обработка событий SignalR на фронтенде
- ✅ Оптимизирована логика отправки уведомлений (не показывать для активного чата)

### Исправлено
- ✅ Проблемы с рассинхронизацией счётчика непрочитанных

## [1.0.0] - 2026-03-27

### Добавлено
- 🎉 Первый стабильный релиз Prime Messenger
- 🤖 Интеграция с AI-ассистентом на базе Ollama
- 💬 Личные и групповые чаты
- 📁 Обмен файлами до 5 ГБ
- 🔐 JWT аутентификация
- 🎨 Темы оформления (фиолетовая, светлая, синяя)
- 📱 Адаптивный дизайн для мобильных устройств
- 🔄 SignalR для обмена сообщениями в реальном времени
- 📊 Отслеживание статуса доставки/прочтения сообщений
- 🤖 Telegram-бот для доступа к Ollama
- 🎭 Системные промпты для AI (глобальные и приватные)
- 🖼️ Поддержка изображений для VL-моделей
- 🔧 Управление моделями Ollama через бота

### Технические детали
- **Backend**: ASP.NET Core 10.0
- **Frontend**: React 19 + Vite + TypeScript
- **БД**: SQL Server LocalDB
- **AI**: Ollama API
- **Real-time**: SignalR
- **Bot**: Python + Aiogram
